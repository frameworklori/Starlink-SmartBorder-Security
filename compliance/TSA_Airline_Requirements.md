# TSA × Airline Integration Requirements
© LORI Framework | compliance/TSA_Airline_Requirements.md

Defines minimal standards for airline cooperation with SmartBorder AI-integrated pre-boarding systems.

---

## 1. Data Handoff
- Airlines must transmit passenger manifest to CBP / TSA 72 hours prior to U.S. entry.
- Flagged individuals from SmartBorder system must be reviewed by TSA checkpoint authority.

## 2. System Compatibility
- Boarding systems must allow RFID + biometric cross-reference (non-mandatory opt-out option).
- Data API must support one-way verification logs with hashed IDs.

## 3. Passenger Rights
- All passengers must receive printed notice of sensor field on boarding gate.
- No invasive scanning without visual notification and public signage.

## 4. Refusal Clause
- Airlines refusing to participate may be disqualified from SmartGate terminal use.
