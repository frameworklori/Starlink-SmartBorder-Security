# Threat Model - STRIDE Framework

## Categories
| Threat | Example |
|--------------|----------------------------------------------------------|
| Spoofing | Faked gait embedding injection |
| Tampering | Modify cached edge results before cloud sync |
| Repudiation | Voice data used without proper attribution |
| Information Disclosure | Unauthorized access to biometric logs |
| Denial of Service | Flood of fake alerts from multiple edge nodes |
| Elevation of Privilege | Overprivileged API tokens from third parties |
