# Starlink-SmartBorder System Architecture Overview

Modular multi-node architecture connecting edge detection with cloud intelligence.

---

## Layers:
1. **Sensor Edge Layer** – UWB, thermal, gait, etc.
2. **Uplink Layer** – Starlink or 5G failover
3. **Cloud AI Layer** – Modular alert system
4. **Review Layer** – Human + Legal node
5. **Command Layer** – DHS/FBI only

## Principles:
- No enforcement by AI
- Logs retained max 30 days unless escalated
- Geofenced zones w/ opt-out logic for citizens
