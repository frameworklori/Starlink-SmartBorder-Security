# Edge Node Design Specifications

Each edge node is a self-contained sensor + compute module for remote border zones.

---

## Components:
- Thermal imager
- LIDAR or UWB motion tracking
- Gait pattern AI module
- Voiceprint optional add-on

## Power Profile:
- Solar primary / battery backup (12h)
- Sleep-wake mode configurable

## Security:
- Encrypted firmware + tamper-proof casing
- Remote kill-switch enabled
