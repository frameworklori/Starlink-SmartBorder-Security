## Report Types:
1. **Public Risk Reports**
- Total alerts, breakdown by type
- False positives vs. confirmed escalations
- Geo heatmaps (without identity)

2. **Oversight Reports**
- Override count by agency
- Escalation justification summaries
- Algorithm patch logs

3. **Incident Logs (Redacted)**
- Description of all confirmed abuses
- Response taken
- Public-facing timeline of resolution

---

## Frequency:
- Monthly (public dashboard)
- Quarterly (PDF + API)
- Real-time (flagged incident tracker)
