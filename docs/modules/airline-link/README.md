
# Airline-Link Integration Modules
© LORI Framework | Internal Module Overview

This directory defines how the Starlink-SmartBorder system interfaces with airline carrier systems and TSA/CBP pre-board authorization workflows.

---

## Files:
- `Carrier_API_Spec.md` – API schema for manifest submission and response handling
- `Joint_Incident_Workflow.md` – Protocol for shared risk/incident management
- `NoFly_Hold_Queue.md` – Temporary holding queue logic for escalated passengers

All modules comply with IATA data schemas and U.S. TSA pre-boarding regulations.
