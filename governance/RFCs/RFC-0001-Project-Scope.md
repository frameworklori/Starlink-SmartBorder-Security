# RFC-0001: Project Scope – Starlink-smartBorder-Security

## Objective
Define the intended scope of border-related AI surveillance, communication, humanitarian routing, and modular integrations with Starlink.

## Components Covered
- Edge detection nodes (thermal, gait, voice)
- Starlink D2C (direct-to-cell) links
- NGO / Civilian route response triggers
- ZTV-compatible alerting

## Non-Goals
❌ Domestic surveillance
❌ Political protest detection
❌ Law enforcement automation without human-in-loop
