# Contributing Guidelines

## How to Contribute
1. Fork this repository.  
2. Propose edits via Pull Request (PR).  
3. Keep tone **analytical and neutral**.  

## Style
- Write in **English** with optional **Chinese translation**.  
- Avoid partisan language or political bias.  

## Responsibility Reminder / 責任提醒
Contributors provide **blueprint ideas only**.  
Policy adoption responsibility belongs to **U.S. elected decision-makers**.  

貢獻者僅提供 **藍圖構想**，政策責任歸屬 **美國人民選出的決策者**。
